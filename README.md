# **Steps to install and run project**

**1. Create .env file** <br/>
**2. Generate token https://github.com/settings/tokens** <br/>
**3. Create same strucutre as .env.template** <br/>
**4. Run yarn && yarn start** <br/>
**5. Enjoy** <br/> <br/>
